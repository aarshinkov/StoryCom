# StoryCom

Share your stories completely anonymous...

## Overview

Tools used:

1. IntelliJ IDEA Ultimate 2018.2
2. Java 8 & JDK 1.8
3. Maven build tool
4. Spring 5
5. Oracle 12c

____

## Implementation

This is a Spring Web MVC project and with it you can freely share your personal stories with other users.
